## v1.0.0
*Initial release*

### Added
- System-wide keyboard hook for global hotkey detection
- Three cycle modes: **NextWindow**, **NextApp**, **Hide**
- Automatic app launch when target process is not running
- System tray integration with right-click context menu
- Start with Windows toggle
- Run as Administrator toggle
- Self-contained build (no .NET install required)
- Portable mode with local config storage
- Chocolatey package (`ipatalas-appswitcher`)
- Windows 10 and Windows 11 support (x64 and x86)
