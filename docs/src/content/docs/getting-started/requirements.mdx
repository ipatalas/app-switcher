title: System Requirements
description: Supported operating systems, architectures, and runtime requirements for AppSwitcher.
---

AppSwitcher is designed to be easy to run with minimal dependencies.

## Operating System

| Version       | Supported | Notes                                              |
| ------------- | --------- | -------------------------------------------------- |
| Windows 11    | ✅        | Full support, including window border pulse effect |
| Windows 10    | ✅        | 20H1 (build 19041) or newer: Full support (border pulse effect not available)   |
| Windows 8.1   | ❌        | Not supported                                      |
| Windows 7     | ❌        | Not supported                                      |

## Architecture

Both **x64** (64-bit) and **x86** (32-bit) is supported but no dedicated builds are provided per architecture.
The same installer and portable builds work on both architectures.

## .NET Runtime

AppSwitcher is built with **.NET 8.0 WPF**.

If you prefer a smaller download, a framework-dependent build is available,
which requires [.NET 8.0+ Desktop Runtime](https://dotnet.microsoft.com/download/dotnet/8.0)
to be installed separately.

If you want a hassle-free experience without worrying about runtimes, the self-contained builds include the .NET runtime and all dependencies, so you can just download and run without any additional setup.

## Permissions

AppSwitcher runs as a **standard user** by default and does not require administrator rights.

There is one exception: switching to applications running with elevated privileges (e.g. Task Manager)
requires AppSwitcher to also run as Administrator. See
[Running as Administrator](/docs/advanced/admin) for details.
