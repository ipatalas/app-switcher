Already installed AppSwitcher? Here's how to go from zero to switching apps in under two minutes.

## Open Settings

Double-click the AppSwitcher icon in your system tray, or right-click it and choose **Settings**.

## Assign Your First Hotkey

<Steps>
1. Click **Add new hotkey** in the settings window.
2. Click the key combination field and press the keys you want to use (e.g. `Apps+C`).
3. Click **Browse** or type the path to the application you want to switch to.
4. Choose a [cycle mode](/configuration/cycle-modes) — leave it as **NextWindow** if you're unsure.
5. Click **Save**.
</Steps>

<Aside type="tip">
  The `Apps` key (also called the **Menu key**) sits between the right `Alt` and `Ctrl` keys on most keyboards.
  It's rarely used by other apps, making it a great modifier for AppSwitcher hotkeys.
</Aside>

## Test It

Press your hotkey. If the app is already running, AppSwitcher will bring it into focus.
If it isn't running yet, AppSwitcher will launch it for you automatically.

## Suggested Starter Setup

Here's a simple setup to get you going:

| Hotkey     | App               |
| ---------- | ----------------- |
| `Apps+B`   | Browser           |
| `Apps+C`   | Code editor       |
| `Apps+T`   | Terminal          |
| `Apps+E`   | File Explorer     |
| `Apps+S`   | Slack / Teams     |

You can always add, edit, or remove hotkeys at any time from the Settings window.
